# Game Shop — Frontend

React SPA для магазина игровой валюты. Курсовой проект по дисциплине «Технология разработки программного обеспечения», траектория Б.

## Стек технологий

- React 19.2.6
- React Router DOM — маршрутизация
- Axios — HTTP-запросы к API
- Context API — управление состоянием авторизации
- JWT — аутентификация (хранение токенов в localStorage)

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить в режиме разработки
npm start
```

Приложение запускается на http://localhost:3000

Бэкенд должен быть запущен на http://localhost:8000

## Структура проекта
src/
├── components/
│   ├── Navbar.jsx        # Навигационная панель
│   └── PrivateRoute.jsx  # Защита приватных маршрутов
├── contexts/
│   └── AuthContext.jsx   # Контекст авторизации
├── pages/
│   ├── Home.jsx          # Главная страница (каталог товаров)
│   ├── Login.jsx         # Страница входа
│   ├── Register.jsx      # Страница регистрации
│   ├── Profile.jsx       # Профиль пользователя
│   ├── Cart.jsx          # Корзина
│   └── Orders.jsx        # История заказов
├── services/
│   └── api.js            # Axios экземпляр с interceptors
├── App.js                # Маршрутизация
└── index.js              # Точка входа
